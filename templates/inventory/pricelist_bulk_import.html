Instrucciones

  1. Descarga la plantilla con el botón de arriba. Contiene todos los productos activos.
  2. La plantilla incluye las columnas fijas:
    • sku — identificador del producto (no modificar)
    • nombre — nombre del producto (no modificar)
    • precio_compra — precio de costo del producto
    • precio_venta_general — precio de venta estándar
  3. {% if pl %} La columna {{ pl.name }} corresponde al precio de esta lista. {% else %} Cada columna adicional corresponde a una lista de precio. Completa el monto para los productos que apliquen. {% endif %}

  5. Guarda el archivo en formato .xlsx y súbelo.
  6. Los cambios se aplican de forma inmediata fila por fila; las filas con errores se reportan sin detener el resto.
